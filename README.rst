================================================================================
applause - Silly reactions to bug closing
================================================================================

:Authors:
    Scott Torborg (scott@cartlogic.com)
:Version: 0.1

Applause when a bug gets closed. Extra cheering for old bugs.

Quick start::

    $ cat ~/.gitconfig
    [github]
    user = joeuser
    token = deadbeefcafebabe...
    $ pip install applause
    $ applause joeuser/myrepo
